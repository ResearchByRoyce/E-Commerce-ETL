# Pipeline Output

Run `run_pipeline.cmd` on Windows or `python run_pipeline.py` on any platform.

The pipeline creates:

- `row_counts.csv`
- `mart_sales_summary.csv`
- `mart_customer_summary.csv`
- `mart_product_performance.csv`
- `mart_delivery_performance.csv`
- `ecommerce_etl.db` (ignored by Git because it is generated)
